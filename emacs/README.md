# Emacs dotfiles

## Emacs key bindings

| Emacs key | Description           |
| --------- | --------------------- |
| , T       | Show TODO tree        |
| , :       | Set tags              |
| , a       | Show agenda           |
| , c       | Org capture           |
| , d       | Org put deadline time |
| , s       | Org put schedule time |
| C-c ,     | Set priority          |
